오늘 한 일

오늘은 Python 멀티프로세싱 과제와 Docker/AWS 배포 미션을 진행했다.

멀티프로세싱에서는 "Process"와 "Queue"를 사용해 여러 프로세스가 작업을 나누어 처리하는 구조를 구현했다. 배포 미션에서는 Docker 이미지를 빌드하고, ECR에 업로드한 뒤, EC2에서 가져와 실행하는 흐름을 확인했다.

결과 평가

오늘의 핵심은 명령어를 단순히 실행하는 것이 아니라, 각 도구가 어떤 역할을 하는지 이해한 점이다.

"Queue"는 작업을 분배하고 결과를 모으는 역할을 하며, 여러 프로세스가 작업을 병렬로 처리한 뒤 "join()"으로 마무리된다.

배포 과정에서는 Docker는 실행 환경을 이미지로 만들고, ECR은 이미지를 저장하며, EC2는 이미지를 실행하는 서버라는 점을 이해했다.
