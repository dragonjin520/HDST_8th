# w2m5 팀 활동 진행 순서
1. Twitter 감정분석 데이터로 모델 학습
2. 각 팀원의 모델의 정확도 비교 및 분석
    - 유튜브 현대차 영상 댓글을 웹 스크레핑
3. 스크레핑 데이터를 모델에서 넣어서 긍정, 부정 라벨 추가  
4. 가장 긍정적인 차종과 가장 부정적인 차종 선택
5. 긍정적인 차종에서 가장 많이 나온 장점, 부정적인 차종에서 가장 많이 나온 단점 추출
6. 결론


## 1. 모델 학습

### 모델 선정

각 팀원별 모델
용선 : 양방향 lstm
용진 : 단방향 lstm
지원 : TF-IDF + Logistic Regression

학습 데이터 : twitter 감정분석 csv

Train acc, train loss
val acc, val loss
구해서 비교하였음. -> csv 표로 첨부

결과 : 용선 모델
근거 : 

### 댓글 웹스크래핑
차종 n개에 대한 영상 리뷰 서치
차종 별 달린 댓글 800개 웹스크래핑하여 저장


## 3. 라벨 추가
댓글 csv 파일을 모델에 넣어서 긍정, 부정, 중립 판단
중립의 경우 긍정, 부정이 아닌 정보글과 같은 경우를 제외하기 위해서 추가.
학습 당시 데이터에는 라벨이 긍정과 부정만 있기 때문에 threshold 0.4, 0.6로 구분해서 중립 구역을 만들었음.


## 4. 가장 긍정적 차종, 가장 부정적 차종
ioniq 9 : 71.3%
Elantra : 43.5%

## 5. 긍정적 차종의 장점, 부정적 차종의 단점
threshold 0.2, 0.8로 확장시켜서 각각 긍정적/부정적 차종에서 가장 자주 발생하는 단어를 수집하였다.


## 6. 결론
가장 긍정적 차종인 아이오닉 9의 장점 단어와 가장 부정적 차종인 엘란트라(아반떼)의 단점 단어를 토대로 비즈니스적 가치를 창출한다.
