## 생각해볼점
이번 과제에서 함수의 기본값은 "Asia"로 설정되어 있다.
이 구조에서는 Asia가 출력되었을 때, 그것이 실제로 "Asia"라는 값을 인자로 전달해서 출력된 것인지, 아니면 아무 인자도 전달하지 않아서 기본값으로 출력된 것인지 결과만 보고는 구분하기 어렵다.

예를 들어 아래 두 코드는 최종 출력 결과가 같다.

Process(target=print_continent)
Process(target=print_continent, args=("Asia",))

하지만 의미는 다르다.

첫 번째 코드는 인자를 전달하지 않았기 때문에 함수의 기본값이 사용된 것이다.
두 번째 코드는 "Asia"라는 값을 명시적으로 전달한 것이다.

즉, 출력 결과만 보면 두 경우를 구분할 수 없기 때문에, 실제 프로그램에서는 기본값이 사용되었는지 여부를 로그나 추가 메시지로 구분하는 것이 더 명확할 수 있다.
