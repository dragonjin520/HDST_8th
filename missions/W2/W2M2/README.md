W2M2 - Python Multiprocessing Process

학습 목표

이번 과제의 목표는 Python의 multiprocessing 모듈을 사용하여 여러 개의 프로세스를 생성하고 실행하는 방법을 이해하는 것이다.

특히 Process 클래스를 사용하여 각각의 프로세스가 독립적으로 함수를 실행하도록 만들고, start()와 join()을 통해 프로세스의 시작과 종료 대기를 제어하는 방법을 확인한다.

⸻

기능 요구사항

* 대륙 이름을 포함한 문장을 출력하는 함수를 만든다.
* 함수는 별도의 인자가 전달되지 않으면 기본값으로 "Asia"를 사용한다.
* 기본값을 사용하는 실행 1번과, "America", "Europe", "Africa"를 전달하는 실행 3번을 수행한다.
* 모든 프로세스가 종료된 뒤 프로그램이 종료되도록 한다.
