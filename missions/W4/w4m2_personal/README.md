## 요구사항

1. 데이터 적재
* NYC TLC Trip Record Data를 Spark DataFrame으로 불러오기
* 여러 달 또는 여러 해의 데이터를 효율적으로 한 번에 처리하기
* CSV, Parquet 등 서로 다른 파일 형식 지원하기

2. 데이터 정제 및 변환
* 결측값을 제거하거나 적절한 값으로 대체하기
* 승차·하차 시각 등 시간 필드를 표준 timestamp 형식으로 변환하기
* 비정상적인 데이터 제거하기
    * 음수 운행시간
    * 음수 또는 비정상 운행거리
    * 하차 시각이 승차 시각보다 빠른 데이터 등

3. 기본 지표 계산
* 평균 운행시간 계산
* 평균 운행거리 계산
* 계산 결과를 사람이 읽기 쉬운 형태로 출력하고 저장하기

4. 피크 시간대 분석
* 시간대별 승차 건수를 집계하기
* 운행 건수가 가장 많은 시간을 피크 시간대로 정의하기
* 하루 0~23시의 택시 수요 분포 시각화하기
* 운행량이 가장 높은 시간대 강조하기

5. 날씨와 택시 수요 분석
* 해당 기간의 외부 날씨 데이터 추가 확보하기
* 기온, 강수량 등의 날씨 데이터와 택시 수요를 결합하기
* 날씨 조건에 따라 택시 운행량이 어떻게 달라지는지 분석하기
* 상관계수나 조건별 평균 비교 등 통계적 방법으로 결과 검증하기

6. 결과물
* 분석 결과를 CSV 또는 Parquet 파일로 저장하기
* 평균 운행시간·거리, 피크 시간대, 날씨 분석 결과를 모두 포함하기
* 막대그래프나 선그래프 등의 시각화 생성하기
* Jupyter Notebook에서 분석 과정과 결과를 설명하기
* Spark 실행 환경이 올바르게 설정되어 있어야 함
